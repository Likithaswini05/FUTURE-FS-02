# FUTURE-FS-02
CRM Application built using React.js, Node.js, Express.js, and MongoDB. The system allows administrators to manage client leads, track lead status, maintain notes, and organize follow-ups through a simple and user-friendly dashboard.
# CRM Application

A simple Customer Relationship Management (CRM) application built with React.js, Node.js, Express.js, and MongoDB.

## Features

* Add and manage leads
* Track lead status
* Store lead information
* Backend API integration
* MongoDB database support
* Responsive dashboard

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB

## Installation

### Backend

```bash
npm install
node server.js
```

### Frontend

```bash
npm install
npm start
```
